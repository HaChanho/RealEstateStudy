종소세 산정 과정 (배우자 합산 누진). 배우자 근로소득 ₩36,000,000 → 근로소득공제 후 근로만 과세표준 2,385만(산출세액 232만). 여기에 사업소득(실현 세전이익 534만)을 합산하면 합산 과세표준 2,919만(여전히 15% 구간) → 합산 산출세액에서 근로만 산출세액 232만을 뺀 차액 ×1.1(지방세) = 추가 종소세 88만, 한계세율 16.5%. 사업소득이 커지면 합산 과세표준이 5,000만(24%)·8,800만(35%)을 넘으며 구간이 자동 상향됩니다. 본건은 실현이익이 작아 15% 구간에 머무릅니다.

매도가 S(1.84억) ≪ ₩355,000,000이므로 본인 명의(35%) 비교 불요 — 배우자 24% 이하 안전. 마진형(156.4M)이면 실현 세전이익이 커져 세후 +1,232만(§5 민감도표). 수리비 참고 상한 ₩700만(repairCostCeiling)은 리모델링 시 참고치일 뿐 EP70·과세에서 제외(전매용 실수리 300만만 위 과세에 반영).

7. D-day 액션 플랜

✅ 권리 확정 완료 — D-1 액션은 명도 변수 1건 + 입찰가 결정. 등기·임차조사·전입세대조사·GG Tip 4종으로 인수 0·임차인 대항력 부존재 전부 검증됐습니다. 남은 건 점유관계 현장확인(명도비 변수) + 형제사건 12425 [재매각] 참고 + 입찰가 확정뿐.

D-1 확인 (권리는 확정)

  • 현장 점유 확인 — 공실/소유자/이녹스 특정(명도비·기간 변수, 권리 등급 무관)
  • 형제사건 12425 [재매각] 경위 참고(과열 후 자금이탈 신호 → 입찰가 보수화)
  • (선택) 매각물건명세서 PDF "비고/특이사항"란 — 인수 0 완전무결
  • (선택) 네이버 60㎡ 호가·체결 → S=1.84억 재확인

입찰 전 (D-20~D-1)

  • 입찰가 결정 (마진형 156.4M 권장 / 균형 이상 비추천)
  • 혼인신고 완료 확인
  • 매매사업자 등록 준비(배우자·면세)
  • D-1 지지옥션 취하·변경 재확인
  • 입찰보증금 1,407만 준비

낙찰 후 (D+0~D+30)

  • 잔금 전 매매사업자 등록(배우자·면세)
  • 경락잔금대출 실행(LTV80%·~132M)
  • 점유자 명도 협상/인도명령(보수 300~500만)
  • 소유권 이전

매도 (D+90~D+120)

  • 네이버부동산 등록, 중립 1.84억 타겟
  • 4개월 내 체결 목표(단기회전)
  • 막히면 산단 배후 임대전환 fallback(약흑자 버티기)

8. 최종 의견

구조는 안전하나 수익이 얇은, "관망 또는 마진형 저가 한정 입찰" 물건입니다. 권리는 A 검증 완료(등기 11건 전부 말소·인수 0·임차인 대항력 부존재), 자금력도 여유(필요 ~4,146만), 산단 배후 임대전환 안전판도 있습니다. 구조적 NO-GO 사유는 없습니다.

다만 호가 하단(S=1.84억) 대비 70%승률 입찰가(1.656억)의 경매 엣지가 1,840만에 불과해, EP70 +1,134만(▷ Watchlist)·세후 +446만으로 사용자 관심 기준(2,500만@70%)에 크게 미달합니다. 입찰가가 조금만 올라도 안전마진이 빠르게 소진되는 구조입니다.

  • 입찰한다면 마진형 156.4M(낙찰확률 50%) — 세후 +1,232만·보수 매도(1.748억)에도 흑자(+467만). 안전마진을 확보하는 유일한 구간.
  • 균형 165.6M 이상은 비추천 — 보수 매도 시 적자 전환(−382만~−1,136만), 산단 단일 평형 매도 유동성 약점과 결합.
  • 절대 상한 177M(세전 본전선 = Walk-away) 초과 금지.
  • 관망도 합리적 선택 — 마진형은 낙찰확률 50%로 미낙찰 가능성이 높고, 낙찰돼도 수익이 얇음. 더 두꺼운 엣지의 물건을 기다리는 것도 정당.

권리·자금·구조는 확정됐습니다. 남은 건 입찰 여부 결정(관망 vs 마진형 저가) + D-1 점유관계 현장확인(명도비 변수)뿐입니다.
