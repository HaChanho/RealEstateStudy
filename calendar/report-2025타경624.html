D-day 전 권리 검증 (필수)

  • 지지옥션 등기권리표 → 말소기준·선순위 인수 유무
  • 법원임차조사 → 대항력 임차인·전입세대
  • 매각물건명세서 PDF → 인수조건·특수권리
  • 미납관리비·장기수선(21년차) 점검

입찰 전 (D-20~D-1)

  • 네이버 호가·실거래 재확인 → S(1억 7,300만) 검증(현 표본 1건)
  • 입찰가 결정 (마진형 저가 위주 권고)
  • 본인 명의 매매사업자 등록 확인
  • D-1 지지옥션 취하·변경 재확인
  • 입찰보증금 1,099만 준비

낙찰 후 (D+0~D+30)

  • 잔금 전 매매사업자 등록(본인·면세)
  • 경락+신용대출 실행(LTV 100%)
  • 점유자 명도 협상 개시(참고 ~300만, 과세 제외)
  • 소유권 이전

매도 (D+90~D+120)

  • 매도용 최소 보수(~800만, 참고·과세 제외) 후 매물 등록
  • 호가 0건 단지 → 직접 호가 형성·홍보 필요
  • 4개월 내 체결 목표(이자 누증 방지)

8. 최종 의견

판정: ⚪ 관망 / 마진형 저가 한정 입찰. 단, 이 결론은 권리 검증을 전제로 합니다(현재 raw 미검증). 70%승률 입찰가(1억 5,570만)에서는 세후이익이 +740만원(EP70 +1,005만)으로 수익은 나지만 얇고, EP70도 ▷ Watchlist tier에 그쳐 사용자 관심 기준(2,500만@70%)을 크게 밑돕니다.

의미 있는 이익은 마진형(1억 4,705만, 낙찰확률 ~50%)으로 싸게 받을 때만 발생합니다(중립 매도 시 세후 +1,348만). 21년차 노후 + 외곽(병천면) + 호가 0건이라 매도 유동성도 약하므로, 무리해서 가격을 올릴 이유가 없는 물건입니다.

  • 진행한다면 → 마진형 1억 4,705만 이하 저가 입찰만(중립 매도 세후 +1,348만, 보수 매도에서도 +749만)
  • 균형(1억 5,570만)·낙찰형(1억 6,262만) → 수익은 얇아지고(균형 중립 세후 +740만·낙찰형 +223만), 낙찰형 보수 매도는 세후 −562만 → 비권장
  • 절대 상한 → 세전 본전선 1억 6,600만 초과 금지(여기에 실수리·명도 참고 변동비를 실제 지출하면 본전선은 더 낮아짐)
  • 대전제 → D-day 전 권리 직접 검증으로 "인수 0" 확정. 미확정이면 입찰 보류

요약: 저가에 통과되면 의미 있는 이익, 경쟁이 붙으면 미련 없이 관망. 보증금은 미낙찰 시 즉시 반환되므로 마진형 저가 도전 자체의 비용 부담은 거의 없습니다.
